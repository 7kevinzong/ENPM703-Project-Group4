# ENPM 703 Group Project (Group 4) - Detecting Fraudulent Credit Card Transactions

## Data

Currently working with the Kaggle dataset. You can download it from [here](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) or just run below command.

`#!/bin/bash
curl -L -o ~/Downloads/creditcardfraud.zip\
  https://www.kaggle.com/api/v1/datasets/download/mlg-ulb/creditcardfraud`

Extract the zip, and copy the `creditcard.csv` to the `data/` folder (create it if you don't have it).
